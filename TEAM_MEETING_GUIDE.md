**목적**: 한 눈에 전체 현황 파악

**포함 내용**:
- 📦 총 Amazon 제품 수: 451개
- 🎯 Laneige 제품 수: 5개
- 📈 총 랭킹 레코드: 7,462개
- 📱 소셜 미디어 데이터:
  - YouTube: 51개
  - TikTok: 133개
  - Instagram: 67개
- ⚙️ 시스템 상태: 1시간마다 자동 수집 중
- 🎯 이벤트 감지: 트렌드 기반 분석 (12시간 윈도우)

**회의 발표 포인트**:
> "현재 451개 Amazon 제품의 랭킹을 추적하고 있으며, Laneige 5개 제품에 대해 총 7,462개의 시계열 데이터 포인트를 수집했습니다. 소셜 미디어 데이터는 YouTube, TikTok, Instagram에서 총 251개 컨텐츠를 분석하고 있습니다."

**포함 데이터**:

| Lip Glowy Balm | 22위 | -1 ↓ | $28,091 | 4.7★ | 33,949 |
| ... | ... | ... | ... | ... | ... |

**회의 발표 포인트**:
> "Laneige Lip Sleeping Mask는 Lip Care 카테고리에서 18위를 유지하고 있으며, 53,000개 이상의 리뷰로 가장 높은 고객 참여도를 보이고 있습니다."

---

### 📈 3. 랭킹 추이 (Ranking Trends)

**목적**: 시간에 따른 순위 변화 파악

**포함 데이터**:
- 최근 7일간의 시계열 데이터
- 제품별, 카테고리별 순위 변화
- 가격, 평점, 리뷰 수 추이

**Excel 팁**:
1. 이 시트에서 피벗 테이블 생성 가능
2. 제품명 필터링하여 특정 제품 추이만 보기
3. 차트 추가하여 시각화:
   - 제품별 순위 변화 라인 차트
   - 가격 vs 순위 상관관계 스캐터

**회의 발표 포인트**:
> "지난 7일간 Lip Sleeping Mask의 순위는 15위~20위 사이에서 안정적으로 유지되고 있습니다. [차트를 보여주며] 가격 변동이 없었고 리뷰는 꾸준히 증가 중입니다."

### 📱 5. 소셜미디어 (Social Media)

**목적**: 바이럴 컨텐츠 및 트렌드 파악

**포함 데이터**:
- YouTube/TikTok 인기 영상 Top 50
- 조회수, 좋아요, 댓글 수
- 게시 날짜, 채널 정보
- 영상 URL (클릭 가능)

**정렬**: 조회수 높은 순

**회의 발표 포인트**:
> "TikTok에서 Laneige 관련 영상 중 최고 조회수는 250만회이며, 주로 '#laneige', '#laniegelipmask' 해시태그를 사용합니다. YouTube에서는 리뷰 영상이 평균 4.5만 조회수를 기록하고 있습니다."

**Excel 팁**:
- 플랫폼별로 필터링
- 조회수 기준 Top 10 차트 생성
- 게시일 기준으로 최근 트렌드 파악

### 🎯 6. 이벤트 (Events)

**목적**: 감지된 이벤트 현황

**현재 상태**:
```
⚠️ 아직 감지된 이벤트가 없습니다.
- 랭킹 데이터가 12시간 이상 축적되면 트렌드 기반 이벤트 감지가 시작됩니다.
```

**회의 발표 포인트**:
> "이벤트 감지 시스템은 현재 가동 중이며, 12시간 이상의 데이터가 축적되면 트렌드 기반 분석이 시작됩니다. 시스템은 순위 구간별로 차등화된 임계값을 적용하여 유의미한 변화만 감지합니다."

**이벤트 발생 시 포함될 정보**:
- 이벤트 타입: STEADY_RISE, STEADY_DECLINE, RANK_SURGE, RANK_DROP
- 심각도: critical, high, medium, low
- 순위 변동폭
- 가격 변화
- Context 수집 여부 (소셜 미디어 연관성)
- 인사이트 생성 여부 (LLM 분석)

---

### ⚙️ 7. 시스템 설정 (System Config)

**목적**: 현재 시스템 설정값 공유

**포함 내용**:

**데이터 수집 설정**:
- Amazon 수집 주기: 1시간마다
- 수집 제품 수: 100개/카테고리
- 수집 카테고리: 5개

**이벤트 감지 설정**:
- 방식: 트렌드 기반 + 구간별 임계값
- 트렌드 분석 시간: 12시간
- 최소 데이터 포인트: 3개
- 일관성 임계값: 60%

**순위 구간별 임계값**:
- 1-5위 (초상위권): ±2위 변동 시 이벤트
- 6-10위 (상위권): ±3위 변동 시 이벤트
- 11-20위 (중상위권): ±5위 변동 시 이벤트
- 21-30위 (중위권): ±7위 변동 시 이벤트
- 31-50위 (중하위권): ±10위 변동 시 이벤트
- 51-100위 (하위권): ±15위 변동 시 이벤트

**회의 발표 포인트**:
> "시스템은 1시간마다 자동으로 데이터를 수집하며, 순위 변동 감지는 상위권일수록 민감하게 설정되어 있습니다. 예를 들어, Top 5 제품은 2위만 변동해도 이벤트가 감지되지만, 51-100위 제품은 15위 이상 변동해야 유의미한 이벤트로 판단합니다."

## 질문 예상 Q&A

### Q1: 데이터 수집 비용은?

**A**:
- Amazon 스크래핑: 무료 (Playwright 기반)
- 소셜 미디어: 무료 (YouTube API, TikTok Apify)
- 이벤트 분석 (OpenAI): 이벤트당 $0.01-0.03 (한 달 예상 $2-5)
- 총 운영 비용: **월 $5 이하** (대부분 무료)

### Q2: 얼마나 자주 업데이트되나요?

**A**:
- Amazon 랭킹: **1시간마다** 자동 수집
- 소셜 미디어: **수동 실행** (필요 시)
- 이벤트 감지: **실시간** (새 랭킹 데이터 수집 후 즉시)
- 리포트 생성: **언제든지** 최신 데이터로 생성 가능

### Q3: 경쟁사 데이터도 수집하나요?

**A**:
- 현재: Laneige 제품 5개만 집중 추적
- 가능: 카테고리 내 Top 100 제품 모두 수집 중
  - Vaseline, CeraVe, Neutrogena, ELF, Maybelline 등 포함
- 확장: 특정 경쟁사만 필터링하여 별도 리포트 생성 가능

### Q4: 이벤트가 감지되면 어떻게 알림받나요?

**A**:
- 현재: 리포트에서 이벤트 시트 확인
- 계획 (Phase 4):
  - 📧 이메일 알림
  - 💬 Slack 알림
  - 📊 자동 리포트 생성 및 전송
  - 📅 일간/주간 요약 리포트

### Q5: 왜 아직 이벤트가 없나요?

**A**:
- 이벤트 감지는 **트렌드 기반** 분석
- 최소 **12시간**, **3개 이상**의 데이터 포인트 필요
- 현재 시스템 가동 시간이 12시간 미만이면 대기 중
- 내일 회의 시점에는 이벤트가 감지될 것으로 예상

### Q6: 프롬프트 테스터는 무엇인가요?

**A**:
- 팀원들이 다양한 AI 프롬프트를 테스트할 수 있는 **웹 UI**
- 이벤트 발생 시 AI가 생성하는 인사이트 품질 개선 목적
- 6가지 템플릿 제공 (기본, 상세, 마케팅, 경쟁 분석 등)
- Side-by-side 비교로 최적 프롬프트 선정
- **실행**: `python scripts/run_prompt_tester.py`

### Q7: 데이터 정확도는 어떻게 보장하나요?

**A**:
- Amazon: 실시간 HTML 파싱 (Playwright)
- 소셜 미디어: 공식 API 사용 (YouTube) 또는 검증된 도구 (Apify)
- 데이터 검증: 중복 제거, 이상치 필터링
- 백업: SQLite 데이터베이스 + JSON 백업

### Q8: 이 시스템으로 무엇을 할 수 있나요?

**A**:

**현재 가능한 것**:
1. ✅ Laneige 제품 순위 자동 추적
2. ✅ 시간에 따른 랭킹 변화 분석
3. ✅ 소셜 미디어 트렌드 파악
4. ✅ 이벤트 자동 감지 (트렌드 기반)
5. ✅ AI 인사이트 생성 (프롬프트 테스터)
6. ✅ Excel 리포트 자동 생성

**Phase 4 (개발 예정)**:

9. 📈 대시보드 (실시간)
